# Local Data Sources

The files located in this directory are downloaded from external sources to ensure the notebooks run offline without external dependency breakages.

* **us-states.json**: Sourced from the PublicaMundi MappingAPI repository at `https://raw.githubusercontent.com/PublicaMundi/MappingAPI/master/data/geojson/us-states.json`.
* **US_Unemployment_Oct2012.csv**: Sourced from the Folium examples dataset at `https://raw.githubusercontent.com/python-visualization/folium/main/examples/data/US_Unemployment_Oct2012.csv`.
